Buttons

Button consists of four Components Primary, Link, Icon Button, Floating Action.

Solid Buttons

                    
                        <button class="btn btn-primary">Primary</button>
                        <button class="btn btn-danger">Danger</button>

Outline Buttons

                    
                        <button class="btn btn-primary-outline">Primary</button>
                        <button class="btn btn-danger-outline">Danger</button>

Icon Button

                        
                            <button class="btn btn-primary d-flex-center">
                                <img src="...." alt="cart" class="btn-icon"> Add to Cart
                            </button>

                            <button class="btn btn-danger d-flex-center">
                                <img src="...." alt="cart" class="btn-icon"> Delete
                            </button>
